# CronicleInstallTool
Scripts to install Cronicle or Cronicle-edge on Fedora


This utility will install Cronicle or Cronicle-edge and the accompanying Systemd service.

<hr>
<h3> Proposed Features - Short Term </h3>
<ul>
  <li>Install and perform initial configuration of Cronicle or Cronicle-edge.</li>
  <li>Contains scripts, function libraries, config files, and systemd service files.</li>
</ul>
<hr>

<h3> Proposed Features - Long Term </h3>
<ul>
  <li>Future support for CentOS, Ubuntu and Debian.</li>
</ul>
